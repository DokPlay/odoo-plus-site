FAQ

Короткие ответы про установку, версии, оплату и независимость проекта.

Какая версия Odoo основная?

Основной фокус - Odoo 19.0 Community Edition. Кастомная разработка обсуждается для Odoo 17.0, 18.0 и 19.0.

Где английская версия?

Английская версия находится на главном URL /, русская - на отдельном индексируемом URL /ru/.

Где будут платные Pro-модули?

Pro-модули скоро появятся. Каждый выпущенный Pro-модуль получит отдельную Boosty-ссылку.

Это официальный проект Odoo?

Нет. Это независимый проект, не связанный с Odoo S.A.

Этот проект не связан с Odoo S.A. Odoo является товарным знаком Odoo S.A. Все модули разрабатываются независимо для Odoo Community Edition.

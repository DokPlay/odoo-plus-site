FAQ

Short answers for installation, version support, payment links and project independence.

Which Odoo version is the main target?

Primary focus is Odoo 19.0 Community Edition. Custom development can be discussed for Odoo 17.0, 18.0 and 19.0.

Where is the Russian version?

The Russian version has its own indexable URL at /ru/ and its own localized pages.

Where will paid Pro modules be available?

Pro modules are coming soon. Each released Pro module will get its own Boosty access link.

Is this an official Odoo project?

No. This is an independent project and is not affiliated with Odoo S.A.
